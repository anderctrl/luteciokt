# Lutecio bot

A simple Kotlin "kord" discord bot


## Features

- Music
- Simple dashboard
- Easy to expand


## Environment Variables

To run this project, you will need to add the following environment variables

`DISCORD_TOKEN` Obtained from discord.dev

`LAVALINK_URL` The url with port of a lavalink server

`LAVALINK_PASS` The password of the lavalink server

`DASHBOARD_PORT` On wich port should the dashbard be exposed to
